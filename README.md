# 23i0530-23i0790-A Compiler Construction Project


### Keyword Meanings:
- **start**: Entry point of the program.
- **finish**: Termination point of the program.
- **declare**: Used to initialize variables.
- **loop**: Defines a repetitive control structure.
- **condition**: Used for branching (if-statements).
- **output**: Prints a value to the standard output.

### Scanning Rules:
- **Identifiers**: Must start with an Uppercase letter [A-Z], followed by up to 30 lowercase letters, digits, or underscores.
- **Numbers**: Supports signed integers and floats with scientific notation (e.g., -5.5e10).
- **Comments**: Single-line uses `##`, Multi-line uses `#* ... *#`.
